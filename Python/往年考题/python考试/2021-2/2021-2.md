## 第五题
```Python
def func5(s):
    dic = {}
    for i in s:
        if i not in dic.keys():
            dic[i] = 1
        else:
            dic[i] += 1
    lst = []
    for j in dic.keys():
        if dic[j] % 2 == 0:
            lst.append(j)
    lst.sort()
    if len(lst) == 0:
        return -1
    else:
        return tuple(lst)
```

def func8(strn, t):
